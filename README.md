# 小说转一卡剧本 Skill

这是一个用于将小说、网文、大纲、人物设定自动改编成「一卡剧本」的中文 AI Skill / Prompt。

适用于：

- 短剧剧本
- 漫剧脚本
- 分镜脚本
- 有声剧脚本
- 小说改编
- 短视频剧情内容

---

## 功能

本 Skill 可以根据小说内容自动生成：

- 剧名
- 剧本简介 / 大纲
- 人物小传
- 世界观设定
- 分集一卡剧本
- 动作描写
- 对白
- OS 内心独白
- 系统提示
- 面板信息
- 每集结尾钩子

---

## 支持题材

支持以下类型：

- 都市重生复仇
- 豪门婚恋
- 渣男小三
- 商战打脸
- 诡异降临
- 系统流
- 规则怪谈
- 死循环
- 悬疑奇幻
- 古代怪谈
- 男频升级爽文

---

## 使用方法

复制 `skill.md` 里的内容，作为 AI 的系统提示词或角色提示词使用。

然后输入：

```text
请使用「小说转一卡剧本 Skill」改编下面小说。

要求：
1. 自动判断类型。
2. 先生成前10集。
3. 保留主线和人物关系。
4. 每集一个核心事件。
5. 每集结尾必须有钩子。
6. 动作描写要有画面感。
7. 对白短促、有冲突。

【小说正文】
这里粘贴小说内容……
